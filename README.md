# CAO - Chief Administrative Officer Case Management System

## Overview
ASP.NET Web Forms 4.8 application for Government Case Management with:
- Checker/Maker Approval Workflow
- Overdue Case Tracking
- SMS Notifications
- GIGW Compliance
- SQL Server 2018 Database

## Features
- User Authentication (Maker/Checker/Admin)
- Case Creation & Management
- Approval Workflow
- Overdue Tracking & SMS Alerts
- Document Upload
- GIGW Compliant UI

## Technology Stack
- ASP.NET Web Forms 4.8
- C# 7.3+
- Entity Framework 6.0
- SQL Server 2018
- Bootstrap 4 (GIGW Compliant)

## Setup Instructions
1. Create SQL Server 2018 database
2. Run database scripts from Scripts folder
3. Update Web.config with connection string
4. Install NuGet packages
5. Run application

## Database
- SQL Server 2018+
- Connection String in Web.config
